# Images

This directory contains images and screenshots for the Discovery module documentation.

## Suggested Images

- `webui-dashboard.png` - Web UI dashboard screenshot
- `device-list.png` - Device listing interface
- `device-details.png` - Device details modal
- `scan-progress.png` - Scanning in progress

## Image Guidelines

- Use PNG format for screenshots
- Use SVG for diagrams when possible
- Keep file sizes under 500KB
- Include alt text in documentation
