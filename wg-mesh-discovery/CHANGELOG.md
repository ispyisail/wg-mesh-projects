# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-25

### Added

- Initial release
- Multi-method device discovery (ARP, NMAP, mDNS)
- DNS integration with dnsmasq
- Device type identification (printers, NAS, cameras)
- Automatic scanning with configurable interval
- Cross-mesh discovery synchronization
- Export functionality (JSON, CSV)
- Optional web UI for management
