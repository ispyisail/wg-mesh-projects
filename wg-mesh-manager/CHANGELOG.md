# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-25

### Added

- Initial release
- Core mesh management commands (mesh-init, mesh-add, mesh-remove, etc.)
- Automated config generation and distribution
- Peer database management
- Backup and recovery functionality
- Health monitoring and status commands
- Library modules (logging, validation, parsing)
- OpenWrt installation support
