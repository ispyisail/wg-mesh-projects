# Images

This directory contains images and diagrams for the Mesh Manager documentation.

## Suggested Images

- `mesh-topology.png` - Example mesh network topology
- `status-output.png` - Screenshot of mesh-status output
- `health-check.png` - Screenshot of mesh-health output

## Image Guidelines

- Use PNG format for screenshots
- Use SVG for diagrams when possible
- Keep file sizes under 500KB
- Include alt text in documentation
